Source Dependency Libraries
---------------------------

|eddsrecorder| depends on |fastdds| ``fastrtps`` and ``fastcdr`` libraries.
In order to correctly execute the Recorder, make sure that ``fastrtps`` and ``fastcdr`` are properly sourced.

.. code-block:: bash

    source <path-to-fastdds-installation>/install/setup.bash

.. note::

    If Fast DDS has been installed in the system, these libraries would be sourced by default.
